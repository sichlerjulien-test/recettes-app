# Session Kickoff — Meal Planner

Brief court à lire au démarrage de chaque nouvelle conversation, sans reconstruire tout l'historique.

## Contexte projet

PWA Next.js/TypeScript — planificateur de repas avec gestion des allergènes. Stack : Next.js 16, React 19, Supabase, Vitest, Playwright.

## Rituels obligatoires

**Ouverture :** lire `CLAUDE.md` + `AGENTS.md` (gates de cadrage, CI, TypeScript).

**Clôture de session :** `npm run end-session` doit afficher OK avant de fermer la conversation. Échec = session non close.
